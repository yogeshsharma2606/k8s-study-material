# Scenario-Based Questions

## Scenario 1
Pods remain in Pending state.
How would you troubleshoot?

Consider:
- Node capacity
- Resource requests
- Taints
- Affinity
- Events

## Scenario 2
Application returns 503 errors after deployment.

Investigate:
- Readiness probes
- Service selectors
- Ingress
- Endpoint objects

## Scenario 3
Pods repeatedly restart.

Check:
- Logs
- Liveness probes
- Configuration
- Secrets
- Events

## Scenario 4
Design Kubernetes for a microservices platform.

Discuss:
- Deployments
- Services
- Ingress
- Autoscaling
- Monitoring
- Security
- Disaster recovery

Always explain your reasoning rather than jumping directly to commands.
