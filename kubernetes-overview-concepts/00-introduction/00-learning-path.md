# Kubernetes Learning Path

## Objective

This handbook is organized so every chapter builds on previous concepts.

1. Why Kubernetes
2. History
3. Container Orchestration
4. Kubernetes vs Docker
5. Cluster Overview

After completing this section, you should understand *why* Kubernetes exists before learning *how* it works.

## Recommended Reading Order

Read chapters sequentially.

Avoid skipping directly to Deployments, Services, or Helm without understanding the control plane first.
